Dans ce projet, nous allons développer une application 
pour l'enregistrement à une séance de sensibilisation sur la sexualité.

L'application demande à l'utilisateur d'entrer son age.
Si son age est strictement inférieur à 16, 
le système lui retourne le message " Tu es un enfant " 
sinon, il lui dit " Tu es majeur "; dans ce cas 
on lui presenteras un formulaire à remplir.
